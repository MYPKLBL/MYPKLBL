# Deployment Instructions

## Vercel
1. Upload this folder to GitHub or import directly to Vercel.
2. Add environment variables from `.env.example`.
3. Deploy.

## Node hosting
1. Upload folder to server.
2. Run:
   npm install
   npm run build
   npm run start
3. Point your domain to the app.

## Before launch
- Replace placeholder logo and hero image
- Connect Supabase for live data
- Test mobile layout
